Add the following mappings to your local hosts file to enable the sample configs...

# MultiHostRedirector Mappings
127.0.0.1 		redirector1.local
127.0.0.1 		redirector2.local
127.0.0.1 		redirector3.local
127.0.0.1 		destination.local

Use the sample web.config to ensure that the HTTP Handler is registered with IIS
